📖 Como funciona o monitoramento diário da GMSS

O sistema funciona em dois níveis: este painel (acesso rápido a portais) + monitoramento automático diário via Claude.

1

Configure suas palavras-chave

Na aba Configurar Busca, defina os termos que descrevem os serviços da GMSS. Ex: equipamento hospitalar, ambulância, gestão saúde.

2

Monitoramento diário automático

Todo dia às 08:00, o Claude pesquisa licitações nos portais principais e entrega um relatório no chat com urgência por prazo, valor estimado e link para o edital.

3

Busca manual pelos portais

Use a aba Portais para acessar qualquer portal com um clique. Use Links de Busca Rápida para links pré-configurados com seus termos em cada portal.

4

Personalização avançada

Para afinar a busca diária, diga ao Claude: "Filtrar apenas municípios de SP" ou "Apenas licitações acima de R$ 50.000".

5

Cadastro para participar

Para disputar as licitações encontradas, a GMSS precisa de cadastro no SICAF (federal), PNCP, BLL e nos portais estaduais onde atua.

ℹ️
Monitoramento ativo!
Busca automática agendada para todos os dias às 08:00. Para alterar, basta pedir ao Claude na conversa.
